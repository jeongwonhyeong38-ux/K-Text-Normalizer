# K-Text-Normalizer
A Python tool for mapping Korean slangs and typos to natural English expressions

It maps internet slang, common typos, and abbreviations into standardized Korean or English equivalents. This tool is essential for preprocessing "noisy" social media data, chat logs, or community forum posts for NLP and data analysis.🚀 Key FeaturesHigh Performance: Uses optimized regular expression (Regex) compilation for $O(n)$ time complexity.Dictionary-Based: Easily customizable mapping.json for adding new slang or project-specific jargon.Conflict Resolution: Implements "Longest Match First" logic to ensure phrases like "알잘딱깔센" aren't incorrectly broken down into "알잘딱".Zero Dependencies: Runs on Python’s standard library (no heavy pip install required for core logic).
